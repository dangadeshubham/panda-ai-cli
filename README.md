PANDA AI — Command-Line AI Assistant

Built my own AI system from scratch.

Instead of just *using AI tools*, I explored how they actually work —  
and built a **command-line AI assistant** powered by local models.

---

About the Project

**PANDA AI** is a CLI-based AI system inspired by tools like Claude Code.

It runs entirely on **local LLMs**, meaning:
- no API usage  
- no data sharing  
- full control over the system  

---

What it does

•  Runs multiple LLMs locally (22B & 30B models)  
•  Switch between models (P01 & P02)  
•  Image understanding using vision AI  
•  Works entirely in terminal (CLI-based)  
•  Fully offline execution  

---

##  Tech Stack

- Python  
- Ollama (Local LLM runtime)  
- CLI-based architecture
